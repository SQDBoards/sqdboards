Keyboard sizes:

40%

Features: minimal functionality with no Number row, F-key row, NumPad, Navigation arrows and with a few mandatory modificator keys.

60%

Features: better functionality, compared to 40%. Now has Number row, but still with no Numpad, Navigation arrows and F-key row.

65%

Features: basically 60% with Navigation arrows and a few more navigation keys (PgUp/Down, PrtSc, Delete).

75%

Features: 65%, expanded with F-key row and one more navigation key (Pause).

80%

Features: AKA TKL, has all the 100% functionality, apart from NumPad.

100%

Features: basic 100% keyboard, has all keys and NumPad.
